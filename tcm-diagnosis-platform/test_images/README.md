# 测试图片目录

## 用途

此目录用于存放测试图片，用于验证面诊和舌诊功能。

## 命名规范

为了便于测试脚本识别，请按照以下规范命名图片：

### 面诊图片
- `face_01.jpg` - 第一张面诊图片
- `face_02.png` - 第二张面诊图片
- 或者使用中文：`面诊_01.jpg`

### 舌诊图片
- `tongue_01.jpg` - 第一张舌诊图片
- `tongue_02.png` - 第二张舌诊图片
- 或者使用中文：`舌诊_01.jpg`

## 图片要求

### 面诊图片
- 清晰的正面照片
- 光线充足
- 面部完整可见
- 无遮挡物
- 分辨率建议：至少 800x600

### 舌诊图片
- 清晰的舌头照片
- 自然伸出舌头
- 舌体完整可见
- 光线充足
- 分辨率建议：至少 800x600

## 测试

运行测试脚本：

```bash
cd backend
python test_api.py
```

测试脚本会自动识别此目录下的图片并进行测试。

## 示例

```
test_images/
├── README.md
├── face_01.jpg          # 面诊测试图1
├── face_02.jpg          # 面诊测试图2
├── tongue_01.jpg        # 舌诊测试图1
└── tongue_02.jpg        # 舌诊测试图2
```

## 注意事项

- 请勿上传敏感或隐私图片
- 测试图片仅用于功能验证
- 建议使用公开的测试图片或自己拍摄
