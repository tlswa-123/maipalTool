### 语音识别
- **引擎**: Web Speech API (浏览器原生)
- **优点**: 免费，实时识别

### AI 对话
- **服务**: WorkBuddy API
- **模型**: GPT-3.5 Turbo / GPT-4
- **配置**: 已预配置，开箱即用

## ⚠️ 注意事项

1. **麦克风权限**
   - 首次使用需要允许浏览器访问麦克风
   - 如果没有弹出权限请求，请检查浏览器设置

2. **网络连接**
   - 语音识别和 AI 对话需要网络连接
   - 语音合成也需要网络连接

3. **浏览器兼容性**
   - 推荐使用 Chrome 或 Edge 浏览器
   - Safari 和 Firefox 对 Web Speech API 支持有限

4. **API 限制**
   - WorkBuddy API 可能有使用限制
   - 如果遇到问题，可以尝试其他 AI 服务

## 🐛 故障排除

### 语音识别不工作
- 检查麦克风是否正常工作
- 检查浏览器是否允许麦克风访问
- 尝试刷新页面

### AI 不回复
- 检查网络连接
- 查看浏览器控制台 (F12) 的错误信息
- 检查服务器日志: `tail -f web_app/app.log`

### 语音播放失败
- 检查网络连接
- 尝试更换语音选项
- 检查浏览器是否支持音频播放

## 📞 获取帮助
