# 🌐 AI 语音助手 Web 应用 - 使用指南

## ✅ 应用已成功启动！

访问地址：**http://localhost:5000**

应用已经在后台运行，你可以直接在浏览器中使用！

## 🎯 如何使用

### 1️⃣ 配置 AI 服务（必须）

在使用之前，你需要先配置 AI 服务：

1. 在页面的"设置"面板中，选择 AI 服务提供商
2. 输入对应的 API Key（Ollama 不需要）
3. 点击"保存设置"

**免费选项：**
- **Ollama**（完全免费，本地运行）
  - 安装：`curl -fsSL https://ollama.ai/install.sh | sh`
  - 下载模型：`ollama pull llama2`
  - 启动服务：`ollama serve`

**付费选项（有免费额度）：**
- **智谱 AI**：https://open.bigmodel.cn/usercenter/apikeys
- **DeepSeek**：https://platform.deepseek.com/
- **OpenAI**：https://platform.openai.com/api-keys

### 2️⃣ 语音识别（用户输入）

#### 方式 1: 语音输入（推荐）

1. 点击 **"开始录音"** 按钮
2. 对着麦克风说话
3. 识别的文字会**实时显示**在输入框中
4. 点击 **"停止录音"** 按钮停止

#### 方式 2: 文字输入

直接在文本框中输入文字即可。

### 3️⃣ AI 对话

1. 输入文字后，点击 **"发送给 AI"** 按钮
2. AI 会思考并生成回复
3. 回复会显示在右侧面板

### 4️⃣ 语音合成（AI 语音）

1. AI 回复后，点击 **"播放语音"** 按钮
2. 系统会播放 AI 的语音
3. 点击 **"停止播放"** 可以停止播放

## 🎨 功能介绍

### 语音识别功能

- ✅ **实时识别**：边说边识别，实时显示

- ✅ **多服务支持**：OpenAI、智谱AI、DeepSeek、Ollama

### 语音合成功能

- ✅ **自然语音**：使用 Edge TTS，语音自然流畅
- ✅ **多种声音**：提供多种语音选择
- ✅ **在线播放**：无需下载，直接播放

## 💡 使用技巧

### 1. 提高语音识别准确率

- 在安静的环境中使用
- 对着麦克风清晰说话
- 不要说太快
- 使用标准普通话

### 2. 优化 AI 回复质量

- 设置合适的系统提示词
- 提供清晰的问题
- 使用更高级的模型（如 GPT-4）
- 明确你的需求

### 3. 获得更好的语音效果

- 选择 Edge TTS 引擎
- 试听不同的语音
- 根据内容选择合适的声音

## 🔧 常见问题

### Q1: 无法访问麦克风？

**A:** 请确保：
- 浏览器允许麦克风权限
- 使用支持语音识别的浏览器（Chrome、Edge）
- 在 HTTPS 环境下运行（localhost 可以）

### Q2: 语音识别不准确？

**A:** 尝试：
- 在安静的环境中使用
- 对着麦克风清晰说话
- 使用标准普通话
- 减少背景噪音

### Q3: AI 没有回复？

**A:** 检查：
- API Key 是否正确
- 网络是否连接
- API 配额是否用完
- 查看操作日志了解详细错误

### Q4: 无法播放语音？

**A:** 确保：
- 浏览器支持音频播放
- 音频文件生成成功
- 音量已打开

### Q5: 如何停止服务器？

**A:** 在终端中按 `Ctrl+C`，或运行：
```bash
pkill -f simple_app.py
```
